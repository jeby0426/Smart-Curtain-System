# Smart-Curtain-System
# 🌞 Smart Curtain System

조도 센서를 이용하여 주변 밝기를 측정하고, 측정된 조도에 따라 커튼을 자동으로 개폐하는 스마트 커튼 시스템입니다.

---

## 📖 프로젝트 개요

기존 커튼은 사용자가 직접 열고 닫아야 하는 불편함이 있습니다.

본 프로젝트는 조도 센서를 이용하여 실시간으로 주변 밝기를 측정하고, 측정된 조도 값에 따라 커튼이 자동으로 열리고 닫히도록 구현하였습니다.

또한 이동 평균 필터를 적용하여 센서 노이즈를 제거하고, 안정적인 제어가 가능하도록 설계하였습니다.

---

## 🎯 개발 목표

* 주변 밝기에 따른 자동 커튼 제어
* 실내 조도 환경 자동 관리
* 사용자 편의성 향상
* 저비용 스마트 홈 시스템 구현

## ⚙️ 하드웨어 구성

| 구성 요소        | 역할             |
| ------------ | -------------- |
| Arduino Mega | 전체 시스템 제어      |
| 조도 센서(LDR)   | 주변 밝기 측정       |
| PCF8591 ADC  | 아날로그 신호 디지털 변환 |
| DC 모터        | 커튼 개폐 조정          |
| MDD3A      | 모터 속도 및 방향 제어  |
| 브레드보드        | 회로 연결          |

## 🔧 주요 기능

### 1️⃣ 조도 측정

* 조도 센서를 이용하여 실시간 밝기 측정
* PCF8591 ADC를 통해 디지털 값으로 변환

### 2️⃣ 이동 평균 필터

* 최근 50개의 측정값을 평균 처리
* 노이즈 제거
* 순간적인 빛 변화에 의한 오동작 방지

### 3️⃣ 자동 커튼 제어

* 조도 값에 따라 목표 개방 정도 계산
* 목표 위치와 현재 위치를 비교하여 자동 제어

### 4️⃣ PWM 모터 제어

* `analogWrite()`를 이용하여 모터 속도 제어
* 정방향 및 역방향 회전을 통해 커튼 개폐

---

## 🔄 시스템 동작 순서

```text
조도 측정
      ↓
이동 평균 필터 적용
      ↓
목표 개방도 계산
      ↓
현재 개방도 비교
      ↓
모터 정/역회전
      ↓
목표 위치 도달
      ↓
모터 정지
```

## 📊 핵심 알고리즘

### 조도 값 읽기

* 조도 센서 값을 실시간으로 수집
* ADC를 통해 디지털 데이터로 변환

### 이동 평균 필터

* 최근 데이터 평균 계산
* 센서 노이즈 감소
* 안정적인 제어 수행

### 목표 개방도 계산

* 조도 값을 0~100% 개방도로 변환
* 밝기에 따라 자동으로 개방 정도 조절
